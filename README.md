# Calculadora Colaborativa

## Descripción
Calculadora por consola desarrollada en equipo para practicar
Forks y Pull Requests en GitHub.

## Funcionalidades
- Suma de dos números
- Resta de dos números
- Menú interactivo con validación

## Cómo ejecutar
python calculadora.py

## Tecnologías
- Python 3
- Git y GitHub
